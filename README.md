## Requirements
- **Java 21 or higher**
- Supported OS: Linux, macOS, Windows

---

## Installation

### Download
You can download the latest release from:
https://github.com/PyleMC/Quantum/releases

### Setup
1. Download Java (minimum 21)
2. Make a folder of your server. Put the `Quantum.jar` file and start script files into the folder.
3. Start the server using the start script files.
4. Change the `quantum.yml` file
